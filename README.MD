# офлайн игра "Морской бой"

## Управление

### Во время размещения кораблей

- ЛКМ по своему (левому) полю - разместить корабль,
- Кнопка "Повернуть корабль" - сменить ориентацию корабля (90 градусов).

### Во время боя

- ЛКМ по полю противника - выстрел.

## Правила игры

Игровое поле — квадрат 10×10 у каждого игрока, на котором размещается флот кораблей.

### Размещаются:

- 1 корабль — ряд из 4 клеток («четырёхпалубный»; линкор)
- 2 корабля — ряд из 3 клеток («трёхпалубные»; крейсера)
- 3 корабля — ряд из 2 клеток («двухпалубные»; эсминцы)
- 4 корабля — 1 клетка («однопалубные»; торпедные катера)

При размещении корабли не могут касаться друг друга сторонами и углами.

### Потопление кораблей противника

Игрок, выполняющий ход, совершает выстрел — выбирает клетку.

``` text
1. Если выстрел пришёлся в клетку, не занятую ни одним кораблём, 
то клетка перекрасится в серый, ход переходит протвнику.

2. Если выстрел пришёлся в клетку, на которой размещён корабль,
то клетка перекрасится в красный, ход переходит протвнику.

```

Победит тот, кто первым уничтожит вражекие карабли.
